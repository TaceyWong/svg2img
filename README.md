# svg2img

将SVG转换为PNG、JPEG、WEBP格式图片

**命令行界面及选项**

```shell
svg2img: 将svg转换为png、jpg、webp格式图片

使用: svg2img [-o 输出文件] [-w 宽] [-h 高] [-q 质量] SVG文件

选项:
  -h 高度
        指定输出图片的高度 (default 512)
  -help
        显示当前帮助信息
  -o 文件名
        指定输出图片的文件名 (default "out.png")
  -q 质量
        输出图片的质量 (default 100)
  -w 宽度
        指定输出图片的宽度 (default 512)
```
